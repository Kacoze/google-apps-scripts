# CI Deploy to Google Apps Script

This repository includes a GitHub Actions workflow for secure deployment to Google Apps Script:

- Workflow: `.github/workflows/deploy-gas.yml`
- Trigger: manual (`workflow_dispatch`)
- Targets: `all`, `absence`, `slack`

## 1. Required GitHub Secrets

Add these repository secrets in GitHub:

1. `CLASPRC_JSON`

- Content of your local `~/.clasprc.json` generated by `clasp login`.

2. `GAS_ABSENCE_SCRIPT_ID`

- Script ID for `google-calendar-absence-sync`.

3. `GAS_SLACK_SCRIPT_ID`

- Script ID for `slack-calendar-status-sync`.

## 2. How to Prepare `CLASPRC_JSON`

1. Run locally:

```bash
npm run clasp:login
```

2. Open local auth file:

- `~/.clasprc.json`

3. Copy entire JSON content and save it as secret `CLASPRC_JSON`.

## 3. Run Deployment

1. Open GitHub repo.
2. Go to `Actions` -> `Deploy GAS`.
3. Click `Run workflow`.
4. Choose target:

- `all`
- `absence`
- `slack`

5. Start workflow.

## 4. Security Notes

- Never commit `.clasprc.json` or `.clasp.json`.
- Rotate auth and tokens if compromised.
- Limit who can run workflows in repository settings.
